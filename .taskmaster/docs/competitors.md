## Ключевые рыночные наблюдения

### Зарубежный рынок
- GenAI (LLM) встроен во все платформы для суммаризации и NL-запросов
- Консолидация вокруг комбинации «реал-тайм подсказки + пост-аналитика»
- NICE, Verint, CallMiner — «Большая тройка» для enterprise
- Observe.AI, Balto, Cresta — сильные челленджеры с современной архитектурой
- Самый быстрорастущий сегмент — real-time guidance (Balto, Cresta)
- Цены: $800-3000+/агент/год для enterprise

### Российский рынок
- Объём ~8.7 млрд ₽ (2026), рост 18-30% ежегодно
- Импортозамещение ускорило переход на отечественные решения
- Переход от «Речевая аналитика 1.0» (ключевые слова) к «Revenue Intelligence 2.0» (LLM)
- Реал-тайм подсказки — пока слабо развиты (кроме 3iTech)
- Yandex SpeechSense — самый доступный облачный вход
- Т-Банк TQM — самая быстрая пакетная обработка
- 3iTech — самый зрелый проприетарный стек
- Дефицит решений для SMB с быстрым стартом и доступной ценой
