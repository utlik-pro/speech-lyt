### Регуляторика — критично

- **Закон №99-З о ПДн** (07.05.2021) обновлён **134-З от 16.03.2026**, в силе с 21.03.2026 — снижает нагрузку на операторов ПДн ([neg.by](https://neg.by/novosti/otkrytj/personalnye-dannye-chto-izmenitsya-v-zakonodatelstve/)).
- **Регулятор НЦЗПД** ([cpd.by](https://cpd.by/)).
- **Трансграничная передача:** без разрешения можно в страны Конвенции СЕ 1981 + ЕАЭС. **США (хостинг OpenAI/Anthropic) НЕ входит** → отдельное разрешение НЦЗПД на каждую передачу.
- **ОАЦ-сертификация** (ТР 2013/027/BY + Приказ №66) — фактически обязательна для госсектора и системно значимых банков. **Срок 6-12 мес, бюджет десятки тыс. USD** ([ОАЦ](https://www.oac.gov.by/activity/certification/technical-and-cryptographic-information-protection/certification-of-information-security-tools)).
- **OpenAI/Anthropic API в РБ:** не работают с РФ/РБ с 2022; не принимают белорусские карты; юрлица РБ не получают валидный инвойс — [vc.ru](https://vc.ru/services/2302708-kak-oplatit-anthropic-ai-iz-rossii). Серый импорт через AIAI.BY и подобных не подходит для крупного бизнеса под санкционным комплаенсом.
- **Санкции США (июнь 2024)** + **ЕС** на enterprise-ПО, IT-консалтинг, cloud для управленческого ПО — [Habr](https://habr.com/ru/articles/822679/).

### Каналы продаж в РБ

- **Конференции:** Call Center Day 2026 (Минск) — [ta-aspect.by](https://ta-aspect.by/training/call-center-day-2026-innovacii-menyayushhie-pravila-igry-ai-analitika-i-personal-kontakt-centra), конференция callcenter.by ([программа](https://callcenter.by/press-center/47)).
- **Интеграторы:** callcenter.by, Axatel/Axata, СофтКлуб, Datastream.by, partners Bitrix24/amoCRM.
- **Прямые продажи:** 11 банков системной значимости, 3 телекома, крупнейшие e-commerce, аутсорс-КЦ.

### 7 ключевых выводов по РБ

1. **On-prem Whisper RU без OpenAI/Anthropic = единственное реалистичное УТП** для банков и госсектора РБ. Это не «фича», а необходимое условие выхода на регулируемые сегменты.
2. **Главный enterprise-конкурент — 3iTech с устоявшимся кейсом Альфа-Банк BY.** Лоб-в-лоб не побить; нужно (а) дешевле/быстрее в SMB, (б) современный AI-стек, (в) гибкий on-prem без российской юрисдикции данных.
3. **Локальная конкуренция почти отсутствует** (только Axatel как часть ВАТС). Окно для «белорусского чемпиона».
4. **ОАЦ-сертификация — must-have** для крупных продаж. Инвестиция 6-12 мес и десятки тыс. USD, но без неё закрыто 50+ % выручки.
5. **TAM узкий, конкуренция тоже:** 5-10 крупных контрактов формируют 70-80 % выручки. Стратегия — land-and-expand с 2-3 якорными кейсами + SMB через partner-канал.
6. **Ценообразование:** 30-80 USD/оператор/мес SMB, 30-200K USD проект Enterprise. Ниже российских 3iTech, соответствует покупательной способности РБ.
7. **GTM-приоритет:** (1) Call Center Day 2026 + конференция callcenter.by, (2) пилот с крупным аутсорс-КЦ, (3) партнёрство с СофтКлуб/Datastream для ОАЦ, (4) реферальные программы с Bitrix24/amoCRM партнёрами.

### Главные гэпы рынка

1. **«Долина смерти» 50-200 операторов** — между потолком SMB-SaaS (~60 тыс. ₽) и полом enterprise (1 млн+). Ёмкость **400-900 млн ₽/год** в РФ — это 20-30 % всего рынка речевой аналитики. **Это и есть основной TAM SpeechLyt.**
2. **Real-time подсказки — слепое пятно:** в РФ только 3iTech имеет зрелый real-time guidance. У всех остальных — либо чистый post-call, либо low-level API. Glob тренд (Balto/Cresta/Observe.AI) растёт за счёт real-time.
3. **AI-коучинг и AI-резюме как стандарт** — в РФ массово отсутствует. Окно 2026-2027.
4. **Self-service онбординг 1-7 дней** — гэп между «3 месяца у ЦРТ» и «только продажи у Rechka/SalesAI».

### Регуляторика РФ (контекст)

- **152-ФЗ** (с 1 июля 2025): запрет первичного сбора ПДн через иностранные хранилища. SaaS обязан хоститься в РФ; банки — УЗ-1 ([b-152.ru](https://b-152.ru/hranenie-personalnyh-dannyh-za-granicej)).
- **Реестр Минцифры** даёт право продаж в госсектор, 0 % НДС, приоритет в 44/223-ФЗ. Требует >50 % российского владения и независимости от OpenAI.
- **187-ФЗ КИИ + 58-ФЗ от 07.04.2025** — ужесточение требований технологической независимости КИИ.
- **OpenAI/Anthropic заблокированы** в РФ; доступ только через прокси (ProxyAPI, AITUNNEL, Polza.AI). Для enterprise/гос — обязательны GigaAM/VoiceKit/SpeechKit + GigaChat/YandexGPT.

### Выводы из РФ для SpeechLyt

1. **Главный гэп — mid-market 50-200 операторов** (400-900 млн ₽/год в РФ).
2. **Real-time + AI-коучинг = главный технологический отрыв** от 3iTech (только enterprise) и SMB-игроков (только post-call).
3. **Ценовая гипотеза 15/60/custom тыс. ₽ корректна,** требует: free-tier лидогенерации, per-minute опции, промежуточного тарифа 90-120 тыс. ₽/мес для верхнего mid-market.
4. **Две архитектуры одновременно:** SaaS на OpenAI через прокси для SMB + on-prem на GigaAM/SpeechKit для enterprise/Реестра Минцифры.

### Стратегия монетизации

1. **14-дневный free trial без карты** на Growth (полный функционал, лимит 500 мин). Конкурентное преимущество vs «по запросу демо» у локальных конкурентов.

4. **Скидки за предоплату:** **−5 % квартал, −12 % полугодие, −20 % год.** (Imot даёт −25 %, Bitrix24 −22 % — мы чуть скромнее, чтобы не размывать MRR.)

6. **Add-ons как upsell-двигатель** — цель поднять ARPU с 1 490 до 2 500 BYN на клиенте Growth за 6 мес.
7. **Партнёрская программа:** 20 % реферальная комиссия первый год для интеграторов телефонии (Cloud-PBX, Tegrus, ИТ-партнёры ПВТ). Критично для входа в банки/телеком.
8. **Локальная биллинг-валюта BYN, договор на белорусское юрлицо** (резидент ПВТ — налог 1 % с выручки + 9 % подоходный — [HTP Belarus](https://eor.by/advantages-benefits-htp-belarus/)).
